# ForecastFM

## to get weather code:
1. run Flask on app.py
2. go to http://127.0.0.1:5000/weather/{latitude}/{longitude}

**Dependencies**

**Process**
```
cd frontend/
npm install
npm install --save-dev typescript svelte-preprocess @tsconfig/svelte
npm run dev

cd backend/
pip install --no-cache-dir -r requirements.txt
```
